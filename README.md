### 线程池源码编写
**线程池**是指一种预先创建一组线程的机制，这些线程在应用程序启动时就已经创建好，等待执行任务。当有新的任务需要执行时，线程池会从线程集合中分配一个空闲线程来执行该任务，每次都重新创建和销毁线程。

**线程池优点**  
1.**提高性能:** 创建和销毁线程是开销较大的操作，通过重用现有的线程，线程池减小了开销  
2.**控制并发量:** 线程池允许你限制并发线程的数量，防止资源耗尽  
3.**简化线程管理:** 避免手动管理线程的生命周期，减少代码复杂性  

**线程池结构**  
缓存队列+线程池+主线程
